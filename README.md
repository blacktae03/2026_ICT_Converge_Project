# 머신러닝 구체화

## 개요

센서가 읽는 값 그 자체는 사실 전압이고 실제 기온, 대기압, 받음각, 풍향, 풍속 등은 우리가 가공을 거쳐야 알 수 있음.  
그래서 기온, 대기압, 받음각, 풍향, 풍속 등을 직접 구해서 신경망에 넣는 것이 아니라,  
센서가 읽는 값 그 자체를 넣어서 바로 유추할 수 있게 모델 설계 예정.

여러 특징을 신경망의 **Input**으로 넣으면 결과적으로 어떤 각도가 최적인지 **Output**으로 얻는 것이 우리의 목표.  
그래서 Regression 모델 사용 예정.

## Layer 형태

![신경망 구조도](nn.svg)

입력 데이터는 2차 또는 3차 **다항식 특징(Polynomial Features)**으로 확장 예정.  
이렇게 하면 모든 특징에 대한 각도를 직선이 아닌 다양한 형태로 추론 가능.  

각 은닉층에는 ReLU 활성화 함수를 사용하고, 가중치에 L2 Regularization을 적용하여  
모델이 과하게 데이터에 고착화되는 현상을 방지할 예정.

## 아직 구체화 안된 부분

센서 값을 몇 개를 사용할 지 몰라서 일단 Input Layer의 개수가 확실하지 않음.  
이게 확실해지면, 2차가 나은지 3차가 나은지 판단하는 게 가능할 듯.  
웬만하면 3차 쓸 것 같긴 함.
